# openkai-themes

A port of [zedokai](https://github.com/slymax/zedokai) themes to [OpenCode](https://opencode.ai/), based on the [Monokai Pro](https://monokai.pro/) color scheme.

<img width="1919" height="1096" alt="THEME_SCREENSHOT" src="https://github.com/user-attachments/assets/988db8b9-b655-4fca-bb37-742fa0fb0363" />

## Themes

12 themes in total, each with a unique dark mode. Light mode has only two variations shared across themes — **normal** for the openkai variants, and **filter sun** for the openkai-darker variants.

| Dark | Light variant |
|------|--------------|
| openkai | normal |
| openkai-classic | normal |
| openkai-machine | normal |
| openkai-octagon | normal |
| openkai-ristretto | normal |
| openkai-spectrum | normal |
| openkai-darker | filter sun |
| openkai-darker-classic | filter sun |
| openkai-darker-machine | filter sun |
| openkai-darker-octagon | filter sun |
| openkai-darker-ristretto | filter sun |
| openkai-darker-spectrum | filter sun |

## Installation

See the [OpenCode themes documentation](https://opencode.ai/docs/themes/).
